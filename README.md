# ProgramForGrandmas
A Program that Grandmas Can Use (No Guarente)

# GNU/Linux
Build With GCC
```bash
gcc -D GNU=1 -o bin/GNU_x86_64_0.0.1  src/main.c
```

# Windows
Build with GCC-WSL
```bash
gcc -D WIN=1 -o bin/WIN_x86_64_0.0.1.exe  src/main.c
```

# Mac
Build with GCC
```bash
gcc -D MAC=1 -o bin/MAC_x86_64_0.0.1.app  src/main.c
```
